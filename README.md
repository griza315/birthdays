# Telegram бот для уведомлений о днях рождения (SQLite)

Бот проверяет дни рождения раз в день и отправляет уведомления в Telegram.

## Функционал

- **Автоматическая проверка** каждый день в 09:00
- **Уведомления**:
  - 🎉 За 7 дней до дня рождения
  - 🎁 За 3 дня до дня рождения
  - 🎂 В сам день рождения
- **Хранение данных** в SQLite базе данных
- **Команды управления** через Telegram

## Установка зависимостей

```bash
pip install pytelegrambotapi apscheduler
```

## Настройка

### Переменные окружения

Создайте файл `.env` или установите переменные окружения:

```bash
TELEGRAM_TOKEN=ваш_токен_бота
CHAT_ID=id_чата_для_уведомлений
DB_PATH=birthdays.db  # опционально, по умолчанию 'birthdays.db'
```

### Получение TELEGRAM_TOKEN

1. Откройте @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

### Получение CHAT_ID

1. Добавьте бота в чат/канал
2. Отправьте любое сообщение
3. Используйте @getmyid_bot для получения ID чата
4. Для личного чата с ботом - это ваш user_id

## Использование

### Запуск бота

```bash
python birthday_bot.py
```

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Показать приветственное сообщение |
| `/check` | Принудительная проверка дней рождения |
| `/add <Имя> <Дата>` | Добавить день рождения (YYYY-MM-DD или DD.MM.YYYY) |
| `/list` | Показать все записи |
| `/delete <ID>` | Удалить запись по ID |
| `/status` | Показать статус бота |

### Примеры

```
/add Иван 1990-05-15
/add Мария 15.06.1985
/list
/delete 1
```

## Структура базы данных

Таблица `birthdays`:
- `id` - уникальный идентификатор
- `name` - имя человека
- `birth_date` - дата рождения (DATE)

## Работа планировщика

Бот использует APScheduler для автоматической проверки каждый день в 09:00 по времени сервера.

## Требования

- Python 3.7+
- Библиотеки: pytelegrambotapi, apscheduler
